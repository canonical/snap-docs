# intel-mei-interface

`intel-mei` enables access to the Intel MEI management interface.

Details about Intel MEI can be found here:
https://www.kernel.org/doc/Documentation/misc-devices/mei/mei.txt

**Auto-connect**: no

Requires snapd version 2.38+.
